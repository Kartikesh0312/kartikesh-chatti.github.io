Brain Tumor Detection using CNNs (April 2023)

Developed a Convolutional Neural Network (CNN) model to classify brain tumor grades using MR Spectroscopic Imaging (MRSI) data. Utilized a dataset of 3,533 MRI scans (2,934 tumorous, 599 non-tumorous) from Kaggle, preprocessed by resizing to 128x128x3, normalizing pixel values, and addressing class imbalance. Applied pre-trained models (ResNet50, InceptionV3, Xception) with transfer learning and data augmentation to overcome limited labeled data. Achieved reliable classification through Stratified K-Fold Cross Validation (5 folds, 4,692 training images, 1,173 validation images per fold). Evaluated performance using accuracy, precision, recall, F1 score, and AUC-ROC metrics.

Tools: Python, TensorFlow, Keras, NumPy, OpenCV

Skills: Machine Learning, Deep Learning, Image Processing, Data Preprocessing, Transfer Learning
